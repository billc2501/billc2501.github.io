Notable Project: Food Ordering and Rating API

Click here to view

In this personal project, I used JavaScript, HTML, and CSS to create two servers. I created one server to allow users to input in the restaurant name and the order through the use of a survey and with the other server, I was able to take in order and store it in a MOngoDB collection. Afterwards, the server would fetch everything in that collection and send data back allowing users to view previous orders. I also developed routes which allowed user to submit food ratings.
